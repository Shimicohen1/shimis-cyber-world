Anthropic has launched Claude Security, a new tool aimed at enhancing Application Security (AppSec) by scanning code for vulnerabilities and suggesting fixes. LΣҒΔ𝕽ΩLL 🇮🇱 reports that this isn't just another static application security testing (SAST) tool. Instead, it attempts to mimic a human security researcher by tracing data flow across files and understanding inter-component communication to identify complex vulnerabilities that traditional scanners often miss.

The tool focuses on critical issues such as injection flaws, authentication bypasses, memory corruption, and logical bugs spanning multiple code layers. LΣҒΔ𝕽ΩLL 🇮🇱 notes that Claude Security is currently available in public beta for Enterprise customers, with Team and Max tier access expected in the future.

This development signifies a shift where AI, which increasingly generates code, is now also being leveraged to hunt for the very bugs it might introduce. For defenders, this means a potential new avenue for proactive vulnerability management, moving beyond signature-based detection to more context-aware analysis.
