🚨 CVE-2026-33477 FileRise is a self-hosted web-based file manager with multi-file upload, editing, and batch operations. In versiosn 2.3.7 through 3.10.0, the file snippet endpoint /api/file/snippet.php allows an authenticated user with only readown access to a folder to retrieve snippet content from files uploaded by other users in the same folder.

## Why It Matters

- Affects Apple ecosystem — update macOS/iOS devices.
- New vulnerability disclosed — verify if your stack is exposed.
