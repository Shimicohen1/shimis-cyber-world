🚨 CVE-2026-34401 XML Notepad is a Windows program that provides a simple intuitive User Interface for browsing and editing XML documents. Prior to version 2.9.0.21, XML Notepad does not disable DTD processing by default which means external entities are resolved automatically. This issue has been patched in version 2.9.0.21. 🎖@cveNotify Links: - - 2.9.0.21 - 2.9.0.21 ---
