🚨 CVE-2026-33986 FreeRDP is a free implementation of the Remote Desktop Protocol. Prior to version 3.24.2, in yuvensurebuffer() in libfreerdp/codec/h264.c, h264->width and h264->height are updated before the reallocation loop. If any winpralignedrecalloc() call fails, the function returns FALSE but width/height are already inflated. This issue has been patched in version 3.24.2.
