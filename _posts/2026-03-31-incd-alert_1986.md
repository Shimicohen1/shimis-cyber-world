The Israel National Cyber Directorate (INCD) has reported a significant increase in password spray attacks targeting Israeli organizations utilizing Microsoft 365 cloud services. This advisory aims to familiarize organizations with this attack method and outline effective mitigation strategies.

Password spraying is a brute-force technique where attackers attempt a small number of common passwords against a large number of user accounts. This method is often employed to bypass account lockout policies, as each account is only attempted with a few passwords before moving on. The INCD's alert highlights the growing threat of these attacks, which can lead to unauthorized access and compromise of sensitive data within Microsoft 365 environments.
